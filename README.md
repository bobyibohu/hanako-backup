# 究极备份

🛡️ Hanako 官方技能 - 一键备份与恢复

## 这是什么？

让 AI 助手具备一键备份和恢复能力，防止改崩。

## 快速开始

### 1. 安装技能

把 `究极备份.txt` 的内容发给 Hanako，等它安装完成。

### 2. 使用方法

| 指令 | 说明 |
|------|------|
| 「帮我备份」 | 一键备份当前目录 |
| 「恢复今天的备份」 | 恢复今天 |
| 「恢复最近的备份」 | 恢复最近 |
| 「有哪些备份」 | 查看备份列表 |

### 3. 终端恢复（Hanako打不开时）

```bash
~/Desktop/hanako backup/2026-03-20/restore.sh
```

把日期改成你实际备份的日期。

## 文件说明

```
├── 究极备份.txt          # 安装文件，发给 Hanako
├── 究极备份介绍.html     # 介绍页面（可分享给朋友）
├── README.md             # 说明文档
└── 2026-03-20/          # 备份示例
    ├── hanako_full_backup_2026-03-20.tar.gz
    └── restore.sh        # 一键恢复脚本
```

## 备份位置

```
~/Desktop/hanako backup/
└── YYYY-MM-DD/
    ├── backup.tar.gz
    └── restore.sh
```
